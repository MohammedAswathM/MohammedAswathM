## Experience & Research Depth

**AI Engineer Intern, September AI**  
**Nov 2025 - Apr 2026 | R&D: AI memory, ontology systems, and agent infrastructure**

Worked on long-horizon memory and ontology systems for AI agents: how an assistant can remember, retrieve, update, and reason over personal and business context across many conversations.

| Area | Work I Contributed To |
|---|---|
| **AI memory systems** | Researched and implemented personal-ontology memory layers covering working memory, episodic memory, knowledge, identity, and social graph retrieval. |
| **Ontology engineering** | Worked on business and personal ontology systems using Neo4j, structured schemas, governed relationships, and graph-native retrieval patterns. |
| **Agentic workflows** | Built and studied planner-executor-verifier flows, intent routing, tool orchestration, ACT-R-inspired retrieval, and verification stages. |
| **Evaluation & reliability** | Worked with LongMemEval-style evaluation, custom eval harnesses, pytest coverage, latency gates, caching, and cost-aware model routing. |
| **Research translation** | Studied transformer, RAG, GraphRAG, contextual retrieval, memory, and knowledge-graph systems, then converted research ideas into working software. |

Selected outcomes include a personal-ontology memory benchmark improvement from **18% to 94% on LongMemEval**, a Neo4j ontology layer with **35 concepts and 180+ governed relationships**, and production-oriented infrastructure with caching, observability, secret scanning, and broad automated tests.
